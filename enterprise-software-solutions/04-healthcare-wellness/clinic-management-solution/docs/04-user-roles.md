# 04 - Clinic Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Practice Owner / MD**| Master Access | Clinical decisions, financial oversight, staff performance reviews. |
| **Associate Physician**| Clinical Access | Consultations, prescribing, clinical documentation. |
| **Clinic Manager / Admin**| Operations Access| Scheduling, billing, inventory, reporting. |
| **Medical Assistant / Nurse**| Operational | Vitals intake, rooming patients, simple procedures. |
| **Receptionist** | Limited Admin | Booking appointments, processing payments, checking insurance. |
| **Billing Specialist** | Financial Access| Claim submission, reconciliation, aging report management. |
| **Patient** | Portal Only | Accessing personal records, booking, paying bills. |
| **Lab Interface** | API Only | Automated result submission and status updates. |
