# 08 - Pharmacy UI Pages

## 1. Professional Dispensing Bench
- High-priority Split View: Active Orders (Left), Clinical Verification (Center), Label Preview (Right).
- "Interaction Board" showing color-coded safety alerts (Red=Fatal, Yellow=Warning).
- Integration for scanning hardware directly from the web interface.

## 2. Inventory & Batch Control Center
- Visual grid of the pharmacy layout showing "Low-Stock" shelves.
- "Cold Chain" dashboard with live temperature gauges for refrigerators.
- Mass-edit interface for receiving large wholesale shipments.

## 3. Narcotics Governance Dashboard
- Real-time ticker of controlled substance usage.
- "Discrepancy Alert" screen showing mismatches between physical counts and digital logs.
- Daily "Report Generator" for mandatory law enforcement submissions.

## 4. Patient Interaction & Sales Hub
- Quick-search for "Prescription Ready" check.
- Integrated POS interface for processing co-pays and non-prescription items.
- "Counseling Alert" modal that forces the pharmacist to sign off on patient education.

## 5. Procurement Planning Hive
- Predictive Stock view: "Recommended Reorder" based on historical trends.
- Vendor comparison grid showing current prices and historical delivery speed.
