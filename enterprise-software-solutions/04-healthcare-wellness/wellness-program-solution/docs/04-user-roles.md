# 04 - Wellness Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Program Director** | Master Access | Global policy, vendor management, executive reporting. |
| **Health Coach** | Clinical Access | 1-on-1 coaching, reviewing HRA data, messaging clients. |
| **Corporate HR Admin** | Admin Access | Employee enrollment, managing challenges, viewing anonymized trends. |
| **Employee / User** | Personal Access| Participating in challenges, syncing wearables, viewing rewards. |
| **Insurance Auditor** | Reporting Only | Verifying participation for premium discount reconciliation. |
| **Content Manager** | CMS Access | Uploading educational videos, articles, and meditation tracks. |
| **Support Agent** | Limited Access | Technical troubleshooting and account management. |
