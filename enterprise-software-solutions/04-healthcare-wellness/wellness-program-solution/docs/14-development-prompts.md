# 14 - Wellness Program Solution Development Prompts

## Core wellness & engagement
### Prompt 1: Multi-Dimensional Health Scoring Engine
"Design a PostgreSQL schema for a corporate wellness health scorer. The system should aggregate data from 'Biometric Screenings', 'Physical Activity' (synced from wearables), and 'Health Assessments'. Implement an algorithm that calculates a 'Wellness Score' (0-100) and provides personalized 'Action Plans'."

### Prompt 2: Real-time Wearable Data Sync Hub
"Develop a Node.js service that integrates with common fitness trackers (Apple Health, Google Fit, Fitbit). Implement a 'Sync Engine' that pulls 'Steps', 'Active Minutes', and 'Sleep Hours' per user. Handle rate-limiting and ensure 'Data Privacy' for sensitive biometric information."

## Interaction & Motivation
### Prompt 3: Dynamic Wellness Challenges & Leaderboards
"Create a React component for a corporate wellness challenge. Support 'Team-based Challenges', 'Individual Milestones', and 'Real-time Leaderboards'. Implement a 'Reward Engine' that automatically awards 'Points' and 'Badges' when a goal is achieved."

### Prompt 4: Integrated Wellness Content Hub
"Design an architecture for a 'Library' of wellness content (Articles, Videos, Meditations). support 'Content Tagging' by goal (e.g., Stress Reduction, Weight Loss, Better Sleep). Implement a 'Recommendation Engine' that suggests content based on the user's 'Health Assessment'."

## Coaching & Support
### Prompt 5: Omni-channel Wellness Coaching Hub
"Develop a communication service for interaction between 'Wellness Coaches' and 'Employees'. Implement 'Secure Messaging', 'Video Coaching Appointments', and 'Goal Commitment' tracking. every interaction must be linked to the user's 'Wellness Plan' in a unified timeline."

### Prompt 6: AI-Powered 'Healthy Habit' Recommender
"Develop an AI service that analyzes a user's activity and assessment data. Use an LLM or NLP model to provide daily 'Nudges' and 'Habit Recommendations' (e.g., 'You've reached your step goal 3 days in a row! How about trying a 5-minute meditation today?')."

## Trust, Privacy & Finance
### Prompt 7: HIPAA-Compliant PHI Privacy Vault
"Implement a robust 'Privacy Policy Engine' specifically for health data. Ensure that employers only see 'Aggregated/Anonymized' data while individual PHI (Protected Health Information) is only accessible to authorized medical/wellness staff."

### Prompt 8: centralized Wellness Audit Transaction Vault
"Design a secure, write-once-read-many (WORM) audit log for all health-related interactions. every assessment completion, biometric update, and coach message must be hashed and stored to ensure data integrity for corporate compliance audits."

## Analytics & specialized Features
### Prompt 9: Executive Population Health Dashboard
"Create a data visualization dashboard tracking 'Overall Engagement Rate', 'Top Health Risks' in the organization, and 'ROI on Wellness Initiatives'. provide 'Year-over-Year' comparisons and identify 'High-Impact' departments."

### Prompt 10: AI-Powered 'Risk of Chronic Disease' Predictor
"Develop a predictive model that analyzes historical assessment and screening data. Use a machine learning algorithm to identify employees at 'High Risk' for conditions like Hypertension or Diabetes and recommend 'Preventive Care' pathways."
