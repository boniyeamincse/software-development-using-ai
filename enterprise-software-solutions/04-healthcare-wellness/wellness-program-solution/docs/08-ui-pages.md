# 08 - Wellness UI Pages

## 1. User Health Hub (Mobile App)
- At-a-glance status: "Steps Today," "Calories Burned," and "Active Challenges."
- "Social Feed" showing friends' milestones and community cheers.
- One-tap "Sync Now" button for manual device oversight.

## 2. Interactive Leadboard Center
- Visual ranking lists with progress bars and "Nudge" buttons for friends.
- Historical "Medal Cabinet" highlighting past challenge wins.

## 3. Digital Health Coach Workbench
- Professional interface showing assigned user risks and program adherence.
- Video-integrated chat for 1-on-1 counseling sessions.
- Task manager for follow-up reminders.

## 4. HR Corporate Insights Panel
- High-level charts: "Department Health Participation," "Average Health Score Improvement," and "Incentive Budget Tracking."
- Anonymized "Heatmap" of corporate health risks (e.g., "High Stress in Logistics Dept").

## 5. Rewards & Perk Shop
- Amazon-style catalog for redeeming "Wellness Points."
- Integration showing "Insurance Premium Savings" based on active program standing.
