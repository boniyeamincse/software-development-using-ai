# 11 - Wellness Security & Privacy

## Individual Health Privacy
- **Strict Data Anonymization**: HR admins can only view "Group Data" when there are more than 5 users to prevent individual identification.
- **Zero-Knowledge Coaching Chat**: E2EE for all 1-on-1 counseling sessions.
- **HRA Encryption**: Health assessment data is encrypted with a user-specific key that only they (and their assigned coach) can unlock.

## Platform Integrity
- **Anti-Cheat Logic**: AI algorithms to detect "False Activity" (e.g., phone shakers) in walking challenges.
- **Device Attestation**: Ensuring health data originates from legitimate hardware rather than API simulators.
- **MFA (Multi-Factor Auth)**: Mandatory for all HR and Program Admin accounts.

## Compliance
- **HIPAA/GDPR Compliance**: Ensuring the platform meets international standards for personal health data management.
- **Right to be Forgotten**: One-click "Data Deletion" for employees leaving the company or program.
- **Immutable Audit Trail**: Non-destructive logging of who accessed which aggregated data set.
