# 08 - Telemedicine UI Pages

## 1. Practitioner Video Console
- High-priority Split View: Provider (Corner), Patient (Center), Clinical Chart (Sidebar).
- "Clinical Toolbar" for capturing screenshots of rashes/wounds with one click.
- Floating "Vitals Overlay" showing live data from the patient's wearables.

## 2. Integrated Waiting Room (Patient View)
- Real-time queue position indicator and "Estimated Join Time."
- Pre-consultation checklist: "Test Camera/Mic," "Verify Insurance," "Sign Consent."
- Educational dashboard: Interactive health articles relevant to the visit reason.

## 3. Remote Monitoring Dashboard
- "Red-Flag" ticker: Patients with vital anomalies requiring immediate reach-out.
- Longitudinal graphs: Visualizing BP or Blood Glucose trends across weeks/months.

## 4. Secure Messaging Hub
- Professional, HIPAA-compliant chat interface for patient-provider co-ordination.
- Automated "Out-of-Office" and "Emergency Escalation" routing UI.

## 5. Virtual Provider Cockpit
- Daily schedule view merging virtual and physical encounters.
- "Revenue Recap" showing processed co-pays and telemedicine claim statuses.
