# 04 - Medical Billing Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Revenue Cycle Director**| Master Access | Financial strategy, clearinghouse contracts, executive reporting. |
| **Billing Specialist** | Operations High | Claim scrubbing, manual corrections, denial appeals, ERAs. |
| **Accountant** | Financial Access| Bank reconciliation, tax reporting, practice payouts. |
| **Patient Financial Rep** | Support Access | Assisting patients with payment plans, statements, and disputes. |
| **Compliance Officer** | Audit Access | Reviewing billing logs for fraudulent patterns or HIPAA violations. |
| **Clearinghouse Agent** | API Only | Automated machine-to-machine data exchange. |
| **Patient** | Portal Only | Viewing statements, paying bills, updating insurance info. |
