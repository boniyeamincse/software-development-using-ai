# 04 - Yoga Studio Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Studio Owner** | Master Access | Financial strategy, teacher payroll, brand settings, executive reporting. |
| **Studio Manager** | Operations High | Class scheduling, managing student accounts, retail inventory. |
| **Lead Instructor** | Professional | Teacher training, sequence planning, community moderating. |
| **Teacher** | Teacher Portal | Class roster management, student notes, substitute requests. |
| **Front Desk Karma** | Limited Ops | Student check-in, selling passes, basic retail sales. |
| **Student** | Personal App | Booking classes, buying passes, tracking practice journey. |
| **Brand Partner** | Limited API | Access for external tea or gear vendors to track sales. |
