# 08 - Yoga Studio UI Pages

## 1. Student App Home (Zen Hub)
- Calm, aesthetic interface: Progress ring for "Classes this Month."
- "My Next Class" card with easy "Cancel" or "Invite a Friend" buttons.
- Curated "Teacher Feature" highlighting the instructor of the week.

## 2. Interactive Class Schedule
- Filter-rich grid view (Yin, Vinyasa, Hatha) with calming color schemes.
- "Booking Modal" showing available credits or upsell to a larger class pack.

## 3. Studio Admin Command Center
- Real-time "Studio Pulse": Total students currently in the building and revenue for the day.
- "Teacher Schedule" grid with one-click "Request Sub" functionality.

## 4. Boutique Retail & POS Bench
- Visual grid of gear and tea items with "Quick Charge to Account" buttons.
- "Inventory Alert" flags for items running low.

## 5. Community Bulletin & Feed
- Pinterest-style feed for studio announcements, healthy recipes, and workshop flyers.
- Interactive comments and "Cheers" for student milestones.

## 6. Executive Growth Dashboard
- Key Charts: "Retention Rate by Teacher," "Intro Pass Conversion," and "Revenue per Mat Space."
- "Churn Risk" roster identifying students who haven't visited in 14 days.
