# 09 - Yoga Studio System Workflows

## The Student Onboarding Journey
1. **Purchase**: Student buys an "Intro Special" via the studio website.
2. **App Setup**: Student receives an SMS link to download the app and sign the digital waiver.
3. **Booking**: Student reserves their first class; System sends "What to expect" email.
4. **Arrival**: App detects student in the lobby; Triggers "Welcome" notification.
5. **Check-In**: Front desk clicks one button to confirm the student's mat position.
6. **Follow-Up**: After class, the app asks for a teacher rating and suggests the next class.

## The Substitute Instructor Workflow
1. Teacher X gets sick and clicks "Request Sub" in their portal.
2. System pushes a notification to all internal teachers qualified to teach "Vinyasa."
3. Teacher Y clicks "Claim Class"; System automagically updates the public schedule.
4. Students booked for that class receive an "Instructor Update" notification.
5. Payroll system logs the change to ensure accurate payment at month-end.

## The Workshop Promotion Workflow
1. Studio Owner creates a "Weekend Wellness Intensive" workshop.
2. System identifies students who have taken >10 Hatha classes in the last 3 months.
3. Target list receives a personalized in-app invite and early-bird discount code.
4. Student books; System sends a digital syllabus and preparatory material.
5. Workshop finishes; Participants receive a digital "Certificate of Completion."
