# 11 - Ecommerce Security & Trust

## Financial Security
- **PCI-DSS Compliance**: Use of hosted payment fields to ensure zero exposure to raw card data.
- **Fraud Detection**: AI-based analysis of IP address, billing-shipping discrepancy, and high-velocity orders.

## System Resilience
- **WAF (Web Application Firewall)**: Mitigating SQL injection and scraper bots.
- **Rate Limiting**: Preventing API abuse by unauthorized crawlers.
- **Data Encryption**: AES-256 for customer PII (Personal Identifiable Information).

## Compliance
- **GDPR / CCPA**: Tools for exporting user data or managing cookie consents.
