# 08 - Ecommerce UI Pages

## 1. Product Landing Page (PLP)
- Virtualized lists for smooth scrolling.
- Faceted search (Category, Brand, Price Range).

## 2. Product Detail Page (PDP)
- Dynamic image gallery with zoom.
- Real-time stock availability badge.
- Cross-sell ("Customers also bought") section.

## 3. High-Conversion Checkout
- Single-page flow to reduce friction.
- Saved cards and address selection.

## 4. Merchant Dashboard
- Real-time sales ticker.
- Inventory "Hot Map" showing fast-moving items.

## 5. Vendor Portal
- Self-registration and KYC (Know Your Customer) upload interface.
