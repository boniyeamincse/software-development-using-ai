# 04 - Ecommerce Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Platform Admin** | Full Access | Financial oversight, vendor approval, system settings. |
| **Store Manager** | Operational Access | Product updates, pricing management, order processing. |
| **Vendor (Seller)** | Seller Portal | Managing personal listings, fulfillment, and payouts. |
| **Customer Support** | Support Access | Handling returns, refunds, and customer tickets. |
| **Logistics Partner** | Fulfillment Access | Updating shipping status and delivery tracking. |
| **Customer** | Frontend Access | Buying products, writing reviews, managing profile. |
