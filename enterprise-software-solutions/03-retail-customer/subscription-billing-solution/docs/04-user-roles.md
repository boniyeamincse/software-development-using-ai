# 04 - Billing Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **CFO / Finance Lead** | Master Access | Revenue reporting, tax configuration, setting billing policies. |
| **Billing Specialist** | Elevated Access | Handling manual refunds, adjusting invoices, managing dunning. |
| **Product Manager** | Catalog Access | Creating new plans, managing coupon campaigns. |
| **Customer Success** | Read-Only | Viewing subscription status to assist with support tickets. |
| **Customer** | Self-Service | Managing their own payment info and plan selection. |
| **Automated System** | Machine-Access | Executing recurring charges and revenue recognition logic. |
