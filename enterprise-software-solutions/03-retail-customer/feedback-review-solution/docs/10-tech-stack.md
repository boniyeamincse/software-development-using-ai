# 10 - Feedback Technology Stack

## Core Engineering
- **Backend**: Python (Django) for administrative logic and data processing.
- **Frontend**: React (Admin) & Svelte (Widgets).
- **Communication**: Postmark (Email) & MessageBird (SMS).

## AI & Data
- **NLP Engine**: HuggingFace (Transformers/BERT) or Azure Cognitive Services.
- **Search**: Elasticsearch / Solr.
- **Data Warehouse**: Google BigQuery.

## Infrastructure
- **Object Storage**: AWS S3 (for photos/videos).
- **CDN**: Cloudflare (for fast widget delivery).
- **Containerization**: Docker & Kubernetes (GKE or EKS).
