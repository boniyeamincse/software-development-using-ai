# 09 - Feedback & Review Workflows

## The Collection Lifecycle
1. **Trigger**: System detects a "Delivered" event from the Courier solution.
2. **Solicitation**: Personalized SMS/Email invitation sent with direct link.
3. **Drafting**: Customer provides rating and text; Optional photo upload.
4. **Ingestion**: System processes submission; Runs sentiment and profanity checks.
5. **Publishing**: Content appears on the merchant website and rich snippets update.
6. **Reputation Management**: Staff reviews the feedback; If negative, an automated ticket is created in the Support solution.

## Moderation Workflow
1. User or AI flags a review for moderation.
2. Review status changes to "Hidden - Under Investigation."
3. Moderator evaluates content; "Approves" (re-publishes) or "Rejects" (permanently hides).
4. System notifies the reviewer if content was rejected for policy violations.
