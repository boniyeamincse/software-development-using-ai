# 08 - Feedback UI Pages

## 1. Sentiment Command Center
- "NPS (Net Promoter Score)" gauge and "CSAT" trends over time.
- Word Cloud showing most frequent positive and negative topics.

## 2. Universal Review Inbox
- Unified feed of reviews from all platforms with "Quick Reply" templates.
- Bulk actions: "Mark as Moderated," "Transfer to Support."

## 3. Mobile Review Interface
- Tap-to-rate stars and one-click photo attachment for customers.

## 4. Widget Configurator
- Preview tool for customizing colors and layouts of web social proof components.

## 5. Dispute Resolution Portal
- Clean interface for managers to review flagged content against company policy.
