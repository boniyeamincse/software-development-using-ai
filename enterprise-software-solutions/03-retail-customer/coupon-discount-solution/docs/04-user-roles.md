# 04 - Coupon Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Marketing Director** | Master Access | Defining global discount strategies, budget approval. |
| **Campaign Manager** | Elevated Access | Creating codes, setting rule constraints, monitoring ROI. |
| **Store Associate** | Manual Access | Applying "In-Store Recovery" discounts at the POS. |
| **Auditor** | Read-Only | Reviewing usage patterns for potential fraud or abuse. |
| **Customer** | End-User | Redeeming codes at checkout. |
| **E-commerce System** | Machine-Access | Executing real-time rule evaluations during checkout. |
