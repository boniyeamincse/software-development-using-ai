# 03 - Coupon System Modules

## Promotion Builder (Admin)
A visual, no-code interface for marketing teams to design complex discount rules and campaign parameters.

## High-Speed Rule Evaluator (Engine)
A dedicated microservice that processes the current cart state and evaluates it against all active promotions in milliseconds.

## Code Generator & Vault
A secure storage module that generates and tracks the lifecycle of millions of unique discount codes.

## Integration API (Public)
The bridge that allows the E-commerce checkout and POS systems to request discount calculations.

## Campaign Analytics Portal
Visualized reporting on campaign health, attribution, and profitability.
