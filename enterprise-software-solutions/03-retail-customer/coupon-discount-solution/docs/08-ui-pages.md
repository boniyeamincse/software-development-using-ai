# 08 - Coupon UI Pages

## 1. Promotions Command Center
- Glancable view of "Active vs. Upcoming" campaigns.
- Total "Revenue Driven" vs "Discount Cost" aggregate charts.

## 2. Rule Configuration Wizard
- Step-by-step builder: Select Type -> Define Constraints -> Set Eligibility -> Set Budget.

## 3. Code Generation & Export Tool
- Interface for creating large batches of codes and downloading them as CSV for marketing partners.

## 4. Real-Time Performance Monitor
- Live feed of redemptions as they happen, with geographic heatmaps.

## 5. Fraud Investigation Inbox
- AI-summarized list of suspicious patterns (e.g., "10 redemptions of one-time code from same IP").
