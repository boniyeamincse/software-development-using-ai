# 02 - Coupon Core Features

## 1. Advanced Code Generation
- Bulk generation of unique, alphanumeric one-time-use codes.
- Human-readable public codes (e.g., "SPRING2026").
- External distribution integration (API-based) for email marketing tools.

## 2. Flexible Promotion Types
- **Flat Amount Off**: $10 off total.
- **Percentage Off**: 20% off specific items.
- **BOGO (Buy One Get One)**: And complex variants like "Buy 3, get the cheapest free."
- **Free Shipping**: Geographic or cart-total based.

## 3. Cart-Level Automatic Discounts
- Discounts applied without a code (e.g., "Spend $100 and get free socks").
- Flash-sale scheduling with automated start/stop times.

## 4. Multi-Constraint Rules Engine
- Exclusions: e.g., "Coupon valid for everything EXCEPT Sale items."
- Eligibility: e.g., "Valid ONLY for first-time customers."
- Combinability: Defining whether multiple coupons can be stacked.

## 5. Usage Budgeting & Guardrails
- Total Campaign Cap: Stop the discount after $50,000 has been redeemed.
- Per-Customer Limits: Limit to 1 use per email address.

## 6. Real-Time Tracking & ROI
- Dashboards showing "Redemption Count," "Average Order Value (AOV) with Discount," and "Generated Revenue."
