# 10 - Membership Technology Stack

## Core Engineering
- **Backend**: Node.js or C#.
- **Frontend**: React (Admins) & Next.js (Members).
- **Mobile**: Flutter or Swift (for Wallet integration).

## Specialized Tools
- **Billing**: Stripe Billing or Braintree.
- **IoT Access**: MQTT / AWS IoT Core.
- **Communications**: Twilio (SMS) & SendGrid (Email).

## Infrastructure
- **Database**: PostgreSQL (Relational) + Redis (Cache).
- **Hardware Integration**: ESP32 or Raspberry Pi gateways at physical entry points.
- **Monitoring**: Datadog (uptime and billing logs).
