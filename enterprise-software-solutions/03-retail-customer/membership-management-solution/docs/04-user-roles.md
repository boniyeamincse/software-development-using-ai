# 04 - Membership Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Owner / GM** | Master Access | Financial strategy, plan pricing, overall staff management. |
| **Membership Manager** | Elevated Access | Approving cancellations, managing overrides, sales reporting. |
| **Front-Desk Staff** | Operational | Checking in members manually, handling walk-in signups. |
| **Accountant** | Financial-Only | Reconciling payments, managing taxes, viewing invoices. |
| **Member** | Self-Service | Updating info, paying dues, viewing digital card. |
| **Access Control System**| Machine-Access | Verifying QR/RFID tokens for entry. |
