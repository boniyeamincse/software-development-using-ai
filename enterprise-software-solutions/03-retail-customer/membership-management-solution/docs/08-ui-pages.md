# 08 - Membership UI Pages

## 1. Digital Member Card (Mobile)
- High-contrast QR code display for quick scanning.
- Plan status badge (e.g., "Active - VIP Member").

## 2. Admin Member Directory
- Data-table with filtering by status, plan, and last-visit date.
- Quick-action buttons for "Freeze," "Upgrade," or "Cancel."

## 3. Financial Dashboard
- Metrics for "Monthly Recurring Revenue (MRR)," "Churn Rate," and "Failed Payments Table."

## 4. Attendance Heatmap
- Visual representation of facility usage throughout the day/week.

## 5. Automated Outreach Builder
- Interface for creating email/SMS templates with dynamic tags (e.g., {{member_name}}).
