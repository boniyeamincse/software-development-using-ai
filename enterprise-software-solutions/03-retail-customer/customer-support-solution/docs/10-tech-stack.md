# 10 - Customer Support Technology Stack

## Core Engineering
- **Frontend**: Next.js (Help Center) and React/Redux (Agent Workspace).
- **Backend API**: Python (FastAPI) or Java (Spring).
- **Messaging Queue**: Apache Kafka (for heavy volume).

## AI & Data
- **Search**: Elasticsearch.
- **AI/LLM**: OpenAI or Anthropic API (for summarization and drafting).
- **Analytics**: Grafana + Prometheus.

## Infrastructure
- **Hosting**: AWS (S3, RDS, EKS).
- **Real-time**: WebSockets (via AWS AppSync or Socket.io).
- **Email Gateway**: SendGrid or AWS SES.
