# 08 - Customer Support UI Pages

## 1. Unified Agent Inbox
- Multi-lane view showing "Unassigned," "Assigned," and "Overdue" tickets.
- Split-screen workspace: Ticket conversation on the left, Customer profile on the right.

## 2. Help Center Portal (Customer Facing)
- Clean, searchable interface with "Popular Articles" and "Contact Us" CTA.

## 3. SLA & Real-Time Performance Monitor
- Gauges showing current CSAT, avg response time, and active agent count.

## 4. Workflow Builder
- Drag-and-drop interface for creating "If-This-Then-That" automation rules.

## 5. Live Chat Widget
- Customizable floating bubble for customer websites with pre-chat forms.
