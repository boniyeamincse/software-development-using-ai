# 04 - Customer Support Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Support Manager** | Admin Access | Configuring SLAs, monitoring team performance, advanced reporting. |
| **Senior Agent** | Elevated Access | Handling escalated tickets, approving refunds, mentoring junior staff. |
| **Support Agent** | Operational | Responding to tickets, updating status, creating KB articles. |
| **Knowledge Base Editor**| Content-Specific| Writing and maintaining FAQs and tutorials. |
| **Customer** | Limited Access | Creating tickets, viewing own history, reading public Help Center. |
| **Technical Specialist**| Cross-Team | Responding to technical tickets, collaborating with dev team. |
