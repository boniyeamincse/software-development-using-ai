# 02 - Restaurant Core Features

## 1. Intelligient Point of Sale (POS)
- Split-check and multi-tender support (Cash, Card, Digital Wallets).
- Table-side ordering via mobile tablets with real-time menu sync.
- "Happy Hour" and "Limited Time Offer" automated pricing logic.

## 2. Kitchen Display System (KDS)
- Digital ticket routing based on prep-time (ensuring all items for a table finish together).
- Real-time stock "86-ing" (automatically disabling out-of-stock items in the POS).

## 3. Floor & Table Management
- Visual floor plan builder with real-time table status (Occupied, Dirty, Available).
- Integrated waitlist and reservation system with SMS notifications.

## 4. Multi-Channel Online Ordering
- Direct-to-consumer white-labeled web ordering portal.
- Aggregator integration (API-based) to sync menu changes and ingest external orders.

## 5. Inventory & Recipe Management
- Ingredient-level tracking with "Theoretical vs. Actual" variance reporting.
- Low-stock alerts and automated Purchase Order (PO) generation for vendors.

## 6. Labor & Shift Scheduling
- Digital clock-in/out with photo verification to prevent "buddy punching."
- Forecasting tools that suggest staffing levels based on historic sales data.
