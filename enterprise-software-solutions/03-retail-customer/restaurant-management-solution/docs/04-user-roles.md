# 04 - Restaurant Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Owner / Franchisee**| Master Access | Multi-unit reporting, tax configuration, global menu edits. |
| **General Manager** | Elevated Access | Managing staff schedules, inventory audits, voiding checks. |
| **Head Chef** | Kitchen Access | Menu engineering, recipe management, KDS configuration. |
| **Server / Bartender**| Operational | Taking orders, processing payments, table management. |
| **Expeditor** | Workflow Access | Final QC of orders, bumping tickets between prep stations. |
| **Courier** | External Access | Picking up delivery orders (via Driver App integration). |
