# 10 - Restaurant Technology Stack

## Core Engineering
- **Backend API**: Node.js (Express) or Python (FastAPI).
- **FOH/Tablets**: React-Native (for cross-platform portability).
- **Admin/Web**: Next.js.

## Specialized Data & Tools
- **Databases**: PostgreSQL (Orders) & Redis (Real-time).
- **Printers**: ESC/POS thermal protocol.
- **IoT / Bridge**: Raspberry Pi with custom local sync service.

## Infrastructure
- **Hosting**: AWS or Google Cloud.
- **Monitoring**: Datadog (uptime of physical terminal devices).
- **Payments**: Adyen, Stripe (Terminal), or Toast-integrated API.
