# 04 - Delivery Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Logistics Director** | Master Access | Financial oversight, carrier management, fleet strategy. |
| **Fleet Dispatcher** | Operational | Real-time monitoring, manual re-assignment, exception handling. |
| **Driver** | Mobile App | Executing deliveries, navigating, capturing ePOD. |
| **Warehouse Staff** | Loading Access | Marking orders as "Picked Up" or "Ready for Dispatch." |
| **Customer Support** | Visibility Only | Viewing delivery status to answer customer inquiries. |
| **Customer** | Tracking Access | Viewing live location of their specific order. |
