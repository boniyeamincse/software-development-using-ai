# 10 - Delivery Technology Stack

## Core Engineering
- **Backend API**: Go (Golang) for performance and concurrency.
- **Frontend**: React (Control Center).
- **Mobile**: Flutter (Cross-platform Driver App).

## Geo-Spatial & Logic
- **Map Provider**: Google Maps Platform or Mapbox.
- **Optimization**: VRP solver based on Google OR-Tools.
- **Database**: PostgreSQL + PostGIS.

## Real-time & Infrastructure
- **GPS Ingestion**: MQTT (AWS IoT Core) or high-speed WebSockets.
- **Cache**: Redis Cluster.
- **Hosting**: AWS (EKS, RDS, S3).
