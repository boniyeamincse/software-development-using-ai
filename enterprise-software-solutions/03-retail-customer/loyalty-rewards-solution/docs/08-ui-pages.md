# 08 - Loyalty UI Pages

## 1. Customer Loyalty Dashboard
- Large visual indicator of points (e.g., Progress ring towards next tier).
- "Personalized For You" reward section with suggested redemptions.

## 2. Gamified Challenges Tab
- Progress bars for active challenges and a "Badges Earned" showcase.

## 3. Rewards Marketplace (Catalog)
- E-commerce style interface for browsing and selecting rewards.
- Categories: Vouchers, Physical Items, Experiences, Charitable Donations.

## 4. Rule Builder Dashboard (Admin)
- Visual flowchart for setting up rules like "IF Product Category = Sale THEN Earning = 0.5x".

## 5. Merchant Search & Adjustment Tool
- Internal tool for store staff to look up members by phone/email and view their history.
