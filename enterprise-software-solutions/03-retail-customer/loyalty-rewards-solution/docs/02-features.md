# 02 - Loyalty Core Features

## 1. Multi-Tier Membership
- Customizable tiers (e.g., Bronze, Silver, Gold, Platinum).
- Automated tier progression based on annual spend or point accumulation.
- Tier-specific perks (e.g., Free shipping, early access to sales).

## 2. Flexible Point Engine
- Define custom earning rules (e.g., 1 point per $1 spent, 2x points on birthdays).
- Support for "Earning by Action" (e.g., Points for social shares, reviews, or referrals).

## 3. Real-Time Redemption
- Seamless checkout integration for using points as a currency.
- "Points + Cash" hybrid payment support.
- Specialized "Reward Catalog" for non-monetary redemptions (e.g., exclusive merch).

## 4. Gamification & Challenges
- Limited-time "Streaks" (e.g., "Visit 3 times this month to earn 500 bonus points").
- Skill-based or behavior-based badges and digital collectibles.

## 5. Referral & Viral Growth
- Integrated referral program with dual-sided rewards (Referrer and Referee).
- Social media sharing tools with unique tracking links.

## 6. Surprise & Delight
- AI-triggered "One-Time" rewards for high-value customers showing signs of churn.
- Personalized anniversary and birthday greetings with exclusive offers.
