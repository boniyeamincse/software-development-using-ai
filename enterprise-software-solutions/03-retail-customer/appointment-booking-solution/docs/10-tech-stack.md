# 10 - Appointment Technology Stack

## Core Engineering
- **Frontend**: Next.js (React) or Vue.js.
- **Backend**: Node.js (NestJS) or Go (for performance).
- **Mobile**: Flutter or Swift/Kotlin.

## Specialized Tools
- **Calendar Sync**: Cronofy or Nylas API.
- **Messaging (SMS)**: Twilio or Plivo.
- **Payments**: Stripe (Connect/Terminal) or Square.

## Infrastructure
- **Cache**: Redis (Essential for slot locking).
- **Database**: PostgreSQL (Relational).
- **Monitoring**: Sentry (Error tracking) and Datadog (Performance).
