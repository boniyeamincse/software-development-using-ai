# 08 - Appointment UI Pages

## 1. Multi-Step Booking Widget
- Service Selection -> Staff Selection -> Date/Time Picker -> Payment -> Confirmation.

## 2. Admin Master Calendar
- Drag-and-drop interface for rescheduling appointments.
- Color-coded views by service type or staff member.

## 3. Staff Availability Builder
- Visual grid for setting weekly recurring schedules and one-off vacation days.

## 4. Customer Account Area
- View upcoming visits and one-click "Rebook Last Service" button.

## 5. Daily Ops Ticker
- A dashboard widget showing "Next Client Arriving" and "Today's Total Revenue."
