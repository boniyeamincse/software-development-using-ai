# 04 - Project Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Portfolio Manager** | Master Access | High-level strategy, cross-project budget approval, global reporting. |
| **Project Manager** | Elevated Access| Creating projects, managing timelines, task assignment, budget tracking. |
| **Team Lead** | Managerial | Overseeing specific task groups, managing sub-teams, resource allocation. |
| **Team Member** | Operational | Executing assigned tasks, logging time, updating status. |
| **External Client** | Restricted Access| Viewing progress on milestones, approving deliverables, commenting. |
| **Viewer / Stakeholder**| Read-Only | Monitoring overall project health and timeline status. |
