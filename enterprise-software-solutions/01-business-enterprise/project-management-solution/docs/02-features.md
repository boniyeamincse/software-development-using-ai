# 02 - Project Core Features

## 1. Multi-Methodology Planning
- **Interactive Gantt Charts**: For long-term waterfall planning and dependency tracking.
- **Agile Kanban & Scrum Boards**: For fast-paced, iterative task execution.
- **Customizable Project Templates**: Speed up setup for recurring project types.

## 2. Advanced Task Orchestration
- Nested sub-tasks, checklists, and task dependencies (FS, SS, FF, SF).
- Recurring task automation for daily, weekly, or monthly rituals.
- Custom task statuses and priority levels tailored to your industry.

## 3. Dynamic Resource Management
- **Workload Balancing**: Visual heatmap of team allocation and capacity.
- **Skill-Based Assignment**: Tagging team members with specific skills for smarter task allocation.
- **Time Tracking Integration**: Log time directly against tasks for accurate labor cost analysis.

## 4. Real-Time Collaboration Hub
- Task-level comments with @mentions and notification threading.
- Integrated file management with version control and "Live Annotations."
- Project-wide announcements and shared calendars for milestones.

## 5. Budget & Cost Tracking
- Baseline budgeting vs. actual spend tracking.
- Automated budget alerts when project costs exceed defined thresholds.
- Billable vs. non-billable hour tracking for client projects.

## 6. Business Intelligence & Reporting
- Customizable project dashboards showing health, velocity, and burn-down.
- Automated weekly status reports generated and emailed to stakeholders.
- Cross-project portfolio reporting for executive visibility.
