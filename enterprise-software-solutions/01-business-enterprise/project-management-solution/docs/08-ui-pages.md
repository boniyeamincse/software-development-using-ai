# 08 - Project UI Pages

## 1. Unified Project Dashboard
- Glancable view of "Task Completion %," "Days Remaining," and "Team Morale."
- Activity stream showing the latest comments and file uploads.

## 2. Interactive Gantt Timeline
- Multi-threaded timeline with drag-to-resize and drag-to-link (dependency) functionality.
- "Critical Path" highlight toggle.

## 3. Collaborative Kanban Board
- Customizable columns with WIP (Work In Progress) limits and "Auto-Archive" rules.
- Fast-action card menus for assignment and prioritization.

## 4. Resource Workload Heatmap
- Visual grid showing every team member across the X-axis (Date).
- Color-coded cells: Blue (Under-utilized), Green (Optimal), Red (Overloaded).

## 5. Rich Document Collaboration
- Markdown-powered task descriptions and project wikis.
- Side-by-side file viewer for comparing deliverable versions.
