# 10 - Project Technology Stack

## Core Engineering
- **Backend API**: Node.js (TypeScript) for high-concurrency real-time updates.
- **Frontend**: Next.js (React) with Tailwind CSS.
- **State Management**: Zustand / Redux Toolkit.

## Real-Time & Search
- **WebSockets**: Socket.io or AWS AppSync.
- **Search Engine**: Elasticsearch (for tasks and documents).
- **Communication**: Integrated with Slack and Microsoft Teams via Webhooks.

## Infrastructure
- **Databases**: PostgreSQL (Main) + Redis (Pub/Sub & Cache).
- **Monitoring**: Sentry (Frontend errors) + Datadog (Backend performance).
- **Documents**: AWS S3 with CDN distribution.
