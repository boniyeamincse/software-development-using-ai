# 08 - Time Tracking UI Pages

## 1. Global Time Entry Portal
- Clean interface with "Start Timer" floating action button.
- Daily/Weekly view toggle for reviewing and editing logs.

## 2. Manager Approval Dashboard
- Filterable queue of "Pending Approval" timesheets.
- Visual breakdown of "Billable vs. Non-Billable" hours for the team.

## 3. Project Burn Dashboard
- Real-time "Fuel Gauge" visualization for project budgets.
- Forecasted completion dates based on current velocity.

## 4. Policy Builder
- Interface for defining complex labor rules: "California Overtime" vs. "EU Working Time Directive."

## 5. Mobile Companion App
- Simplified "Tap-to-Track" interface with location-aware project suggestions.
- Push notification center for submission reminders.
