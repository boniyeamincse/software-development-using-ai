# 10 - CRM Technology Stack

## Core Engineering
- **Backend API**: Python (FastAPI) or Node.js.
- **Frontend**: Next.js (React) with Tailwind CSS.
- **Mobile**: Flutter (for the mobile sales portal).

## Communication & Logic
- **Telephony**: Twilio Voice / SendGrid.
- **Workflow**: Temporal.io.
- **Search**: Elasticsearch (for activity logs).

## Infrastructure
- **Databases**: PostgreSQL (Relational) + Redis (Session Cache).
- **Documents**: AWS S3.
- **Identity**: OAuth2 with Microsoft & Google integrations.
