# 08 - CRM UI Pages

## 1. Kanban Pipeline View
- Card-based visualization of all deals sorted by stage.
- Color-coded "Stagnation Alerts" for deals that haven't moved in X days.

## 2. Universal Contact Timeline
- Infinite-scroll feed showing a mix of emails, notes, recordings, and field changes.
- "Hover-to-Listen" for call recordings.

## 3. Executive Sales Leaderboard
- Real-time stack ranking of sales reps by revenue, activity count, and win rate.

## 4. Automation Sequence Builder
- Step-by-step visual tool: Wait 2 Days -> Send Template A -> If Replied -> Stop -> Else -> Call.

## 5. Rich Quote Editor
- Real-time co-editing of product tables and pricing for professional proposals.
