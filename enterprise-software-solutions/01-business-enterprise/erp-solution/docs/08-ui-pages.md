# 08 - ERP UI Pages

## 1. Global Financial Console
- Real-time P&L (Profit and Loss) visualizations and cash-flow monitors.
- Deep-dive interface for general ledger reconciliation.

## 2. Interactive Warehouse Map
- Visual display of stock levels across geographic locations.
- Heatmap showing high-velocity SKUs vs. stagnant inventory.

## 3. Manufacturing Gantt Chart
- Drag-and-drop production scheduler for factory managers.
- Real-time "Line Down" alerts with visual diagnostic indicators.

## 4. Procurement Portal
- Unified vendor management dashboard with historical lead-time charts.
- Bulk PO generation and approval interface.

## 5. Audit & Compliance Log
- Searchable history of every system-wide transaction and user modification.
