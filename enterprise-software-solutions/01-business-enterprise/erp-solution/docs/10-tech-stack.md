# 10 - ERP Technology Stack

## Core Engineering
- **Backend API**: Java (Spring Boot) or C# (.NET).
- **Frontend**: React-Next.js with Enterprise UI (Ant Design or MUI).
- **Automation**: Camunda or Temporal.io for BPM (Business Process Management).

## Data & Intelligence
- **Primary DB**: PostgreSQL (with high-availability replication).
- **Search**: Elasticsearch (for global record discovery).
- **Data Warehouse**: Snowflake.

## Infrastructure
- **Hosting**: Hybrid-Cloud (AWS + On-Premise synchronization).
- **Messaging**: Apache Kafka.
- **Monitoring**: Datadog (Transactional tracking and hardware metrics).
