# 04 - ERP Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **CFO / Controller** | Master Access | Financial reporting, fiscal policy, audit oversight. |
| **Operations Manager** | Elevated Access | Warehouse strategy, vendor negotiations, logistics. |
| **Production Supervisor**| Manufacturing | Shift scheduling, BOM management, quality control. |
| **Purchasing Agent** | Procurement | Issuing POs, managing vendor deliveries. |
| **Inventory Clerk** | Functional | Logging stock arrivals, cycle counting, picking orders. |
| **System Admin** | Configuration | User management, security patching, integration setup. |
