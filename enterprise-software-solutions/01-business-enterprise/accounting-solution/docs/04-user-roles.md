# 04 - Accounting Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **CFO / Finance Lead** | Master Access | Executive reporting, setting fiscal periods, audit approval. |
| **Controller** | Elevated Access | Managing the GL, approving reconciliations, financial closing. |
| **AR Specialist** | Account Access | Invoicing customers, applying payments, handling collections. |
| **AP Clerk** | Expense Access | Processing vendor bills, preparing payments, expense review. |
| **Staff Accountant** | Operational | Journal entries, bank reconciliation, fixed asset tracking. |
| **External Auditor** | Read-Only | Periodic review of ledger history and compliance logs. |
