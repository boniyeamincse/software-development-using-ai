# 08 - Accounting UI Pages

## 1. Professional Financial Dashboard
- High-level charts for "Operational Cash Flow," "Burn Rate," and "Revenue vs. Expense."
- Quick-links to unbalanced transactions or pending bank reconciliations.

## 2. Dynamic General Ledger Explorer
- Searchable data table with infinite scroll.
- Multi-dimensional filters: Filter by Account, Entity, Project, or Date Range.

## 3. Interactive Bank Reconciliation Tool
- Side-by-side view: Bank Statement vs. System Records.
- One-click "Smart-Match" buttons with confidence scores.

## 4. Invoice & Billing Center
- Visual status tracker for all customer invoices (Draft -> Sent -> Viewd -> Paid).
- Interactive age-analysis charts.

## 5. Report Builder
- Drag-and-drop tool to create custom Profit & Loss (P&L) views.
- Export options for PDF, CSV, and XLSX (formatted for Excel).
