# 04 - Compliance Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Chief Compliance Officer**| Master Access | Global risk oversight, policy finalization, board reporting. |
| **Compliance Manager** | Elevated Access | Designing controls, managing audits, investigating incidents. |
| **IT Auditor** | Technical Access | Verifying technical evidence, infrastructure scans, access reviews. |
| **External Auditor** | Restricted Access| Reviewing scoped evidence folders and provided log historicals. |
| **Employee** | Training Access | Completing assigned training, signing policies, reporting incidents. |
| **Risk Owner** | Operational | Managing remediation tasks for their specific department. |
