# 08 - Expense UI Pages

## 1. Unified Employee Dashboard
- Personal "Cloud Wallet" showing processed receipts vs. Draft reports.
- Progress tracker for submitted reports (Manager Approval -> Finance Review -> Paid).

## 2. Mobile Capture Interface
- Camera view with "Auto-Crop" and "Real-time OCR" feedback.
- Simple "Offline Mode" for capturing receipts on planes or remote locations.

## 3. Manager Approval Inbox
- Card-based view of pending reports with "Review Receipt" modal.
- One-click "Approve All" for clean, in-policy reports.

## 4. Policy Builder & Spend Console
- Intuitive interface for Finance to build rules: "If Category = Meal and Amount > $50, require Manager approval."
- Global per-diem management map.

## 5. Executive Spend Analytics
- High-level charts: "Category Breakdown," "Departmental Burn," and "Travel ROI."
- Fraud alert center highlighting duplicate receipts or weekend spend.
