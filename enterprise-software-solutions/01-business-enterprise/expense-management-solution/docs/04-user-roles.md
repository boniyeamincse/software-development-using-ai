# 04 - Expense Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **CFO / Finance Lead** | Master Access | Setting global spend policies, approving high-value reports, audit oversight. |
| **Finance Admin** | Elevated Access | Processing reimbursements, managing card integrations, reconciling accounts. |
| **Approving Manager** | Managerial Access | Reviewing and approving/rejecting team expense reports. |
| **Employee (Submitter)**| Operational | Capturing receipts, creating reports, viewing reimbursement status. |
| **Auditor** | Read-Only | Reviewing historical spend records for compliance and fraud detection. |
| **System Admin** | Configuration | User management, security patching, API integration management. |
