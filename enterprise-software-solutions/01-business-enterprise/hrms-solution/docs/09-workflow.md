# 09 - HRMS System Workflows

## The Onboarding Lifecycle
1. **Trigger**: Recruiter marks candidate as "Accepted."
2. **Setup**: IT automatically notified to provision hardware/email (via integration).
3. **Documentation**: System sends e-signature packet for contracts and handbook.
4. **Integration**: New hire details synced to the ERP/Accounting system for payroll setup.
5. **Launch**: Welcome email sent to team; Onboarding checklists assigned to the new employee.

## Leave Approval Workflow
1. Employee submits a 5-day vacation request in the ESS.
2. System checks leave balance; if insufficient, request is blocked.
3. Notification sent to immediate Manager (Mobile + Email).
4. Manager reviews team calendar (Overlap Check) and approves.
5. Notification sent back to Employee; Attendance system updated for تلك dates.

## Performance Feedback Loop
1. HR triggers "Year-End Review" cycle.
2. Employee submits "Self-Evaluation."
3. Peer feedback requested and collected anonymously.
4. Manager writes final evaluation based on 360-degree data.
5. Final meeting held; ratings synced to compensation module for potential bonus logic.
