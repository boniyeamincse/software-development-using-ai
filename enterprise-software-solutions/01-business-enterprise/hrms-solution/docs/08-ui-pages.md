# 08 - HRMS UI Pages

## 1. Unified Employee Self-Service (ESS)
- Personal "Dashboard" showing remaining vacation days, next paycheck date, and pending tasks.
- "News & Updates" feed for company-wide announcements.

## 2. Interactive Org Chart (Canvas)
- Dynamic, zoomable visualization of the company structure.
- Modal-based details for each employee (Title, Contact, Team).

## 3. Recruitment Funnel (ATS Pipeline)
- Kanban view of candidates moving from "Applied" -> "Screening" -> "Interview" -> "Offer."
- Drag-and-drop resume comparison tool.

## 4. Performance Review Workbench
- Split-screen interface for writing evaluations while viewing the employee's yearly goals and achievements.
- Qualitative sentiment analysis charts for manager feedback.

## 5. Payroll Processing Center
- High-level sanity-check dashboard showing "Net Change in Salary Spend" vs. last month.
- One-click file generation for bank transfers.
