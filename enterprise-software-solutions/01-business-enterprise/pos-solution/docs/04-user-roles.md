# 04 - POS Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Store Manager** | Master Access | Shift overrides, cash drawer closing, local inventory adjustments. |
| **Cashier / Associate**| Operational | Processing sales, handling returns, signing up loyalty members. |
| **Inventory Clerk** | Stock Access | Receiving shipments, performing cycle counts, stock labeling. |
| **Regional Manager** | Reporting Only | Multi-store performance analytics, setting region-wide promotions. |
| **Support Tech** | Hardware Access| Troubleshooting terminal connectivity and device drivers. |
| **Customer (Self-Service)**| Kiosk Access | Browsing catalog, self-checkout, digital payment. |
