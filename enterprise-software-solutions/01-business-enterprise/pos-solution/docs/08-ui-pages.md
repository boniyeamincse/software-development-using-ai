# 08 - POS UI Pages

## 1. The Main Register Interface
- Large-target touch grid for item selection.
- Clear "Order Summary" sidebar showing real-time totals and tax.
- Fast-action "Pay" and "Loyalty" buttons.

## 2. Integrated Payment Modal
- Clean, focused interface showing "Pending Payment" while hardware processes the card.
- Options for digital signature and receipt selection.

## 3. Local Inventory Tracker
- Optimized view for "Shelf-Side" stock checks and price verifications.
- Barcode scanning mode for rapid inventory lookups.

## 4. Shift & Reconciliation Bench
- Guided "Cash Count" wizard for closing out registers.
- Visual discrepancy alerts if "Expected" vs "Actual" cash don't match.

## 5. Store Insights Dashboard
- Glancable charts: "Top Selling Items," "Hourly Sales Trends," and "Average Basket Size."
- Real-time "Stock Low" priority list.
