# 08 - University UI Pages

## 1. Multi-Dimensional Dashboard
- Heatmaps of campus resource usage.
- Personal "Academic Progress" bar for students.

## 2. Research Repository
- A public-facing search portal for institutional research output.

## 3. Course Registration "Shopping Cart"
- Interactive UI where students can drag-and-drop potential courses to see timetable compatibility before committing.

## 4. Faculty Grade Submission
- Bulk upload tools for large lecture halls (500+ students).

## 5. Alumni Networking Hub
- Map-based view of alumni locations and industries for mentoring opportunities.
