# 10 - University Technology Stack

## Core Technologies
- **Frontend**: Next.js (for SSR performance on large catalogs).
- **Backend**: Go (for high-concurrency registration events).
- **Database**: PostgreSQL with PostGIS (for campus mapping features).
- **Caching**: Redis.

## Integration & Standards
- **LTI (Learning Tools Interoperability)**
- **SAML / OAuth2**
- **S3 (Object Storage)**

## Observability
- **Prometheus & Grafana**: Tracking system load during high-concurrency events (like Results day).
