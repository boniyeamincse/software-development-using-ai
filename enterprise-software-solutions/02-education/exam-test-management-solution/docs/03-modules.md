# 03 - System Modules

## Paper Setter Module
Collaborative environment for faculty to draft, review, and approve exam papers. Includes an "Auto-Generate" feature based on difficulty weightage.

## Controller of Exams (COE) Module
The administrative dashboard for scheduling exams, assigning invigilators, and managing hall tickets (admit cards).

## Student Portal (Assessment Interface)
A distraction-free interface where students take exams. Includes system checks (camera, mic) before the start.

## Invigilator Module
Real-time dashboard for supervisors to monitor students, flag suspicious behavior, and broadcast announcements.

## Evaluator Module
Interface for manual grading of descriptive answers with built-in plagiarism checks and annotation tools.

## Analytics & Insights
Generates performance reports, class-wise averages, and psychometric analysis of questions (validity/reliability).
