# 02 - Core Features

## 1. Advanced Question Bank
- Categorization by subject, topic, and difficulty level.
- Support for LaTeX for mathematical formulas and diagrams.
- Version control for question updates.

## 2. Multi-Format Assessment
- **Computer-Based Testing (CBT)**: Online exams with timed sections.
- **OMR Integration**: Automatic scanning and marking of physical answer sheets.
- **Oral/Viva Exams**: Video-conferencing integration for viva-voce.

## 3. AI-Powered Proctoring
- Facial recognition and attention tracking.
- Detection of browser-switching or split-screen attempts.
- Audio analysis for unauthorized communication.

## 4. Result Processing
- Instant results for objective tests.
- Rubric-based grading for descriptive papers.
- Automated moderation and standard deviation analysis.

## 5. Secure Delivery
- End-to-end encrypted question paper delivery.
- One-time password (OTP) access for invigilators.
