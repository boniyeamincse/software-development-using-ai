# 04 - Exam Roles & Permissions

| Role | Access Level | Responsibilities |
| :--- | :--- | :--- |
| **COE (Controller)** | System Admin | Global scheduling, grade finalization, system logs. |
| **Paper Setter** | Subject Access | Drafting questions, creating blueprints, reviewing papers. |
| **Invigilator** | Live Session Access | Real-time monitoring, attendance, flagging incidents. |
| **Evaluator** | Grading Access | Scoring descriptive answers, applying rubrics. |
| **Student** | Assessment Only | Taking exams, viewing personal results/feedback. |
| **IT Staff** | Infrastructure Access | Server maintenance, secure browser configuration. |
