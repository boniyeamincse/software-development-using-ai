# 09 - Exam System Workflows

## The Exam Lifecycle
1. **Creation**: Paper setters draft questions; COE reviews and finalizes the "Paper Set."
2. **Scheduling**: System assigns hall tickets and invigilators; Sends calendar invites to students.
3. **Execution**: Students login via Secure Browser; AI Proctoring monitors the session.
4. **Grading**: Instant auto-marking for MCQs; Manual routing for descriptive answers.
5. **Vetting**: Results reviewed for outliers; Grading curves applied if necessary.
6. **Release**: Grades published to Student Portals; PDF report cards generated.

## Proctoring Incident Workflow
1. AI detects unauthorized person in frame.
2. Incident logged with timestamp and image snippet.
3. Invigilator receives alert; views live feed.
4. Invigilator gives a "Warning" or "Suspends" the exam.
5. COE reviews the recording for final disciplinary action.
