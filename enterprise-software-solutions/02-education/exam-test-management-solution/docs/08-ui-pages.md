# 08 - Exam UI Pages

## 1. Lockdown Exam Interface
- Full-screen mode with disabled ESC and Alt-Tab keys.
- On-screen calculator (if permitted).
- Progress indicator and "Time Remaining" clock.

## 2. Invigilator Command Center
- Thumbnail grid view of all active student cameras.
- Chat interface for broad institutional announcements.

## 3. Evaluator Annotation Board
- Side-by-side view of student answer and grading rubric.
- Digital "Red Pen" for marking descriptive answers.

## 4. Result Dashboard
- Graphic breakdown of performance (Pie charts of grade distribution).
- "Topper List" and "Subject-wise Analytics."
