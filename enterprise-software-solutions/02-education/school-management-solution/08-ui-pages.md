# 08 - UI Pages & Workflows

## 1. Dashboard (Role-specific)
- **Admin**: Revenue charts, daily attendance summary, staff status.
- **Teacher**: Today's schedule, pending grading notifications.
- **Student**: Upcoming exams, assignment deadlines.

## 2. Student Directory
- Searchable list of all students with quick-action buttons for editing or viewing performance.

## 3. Gradebook
- Matrix view for teachers to enter marks for any given class/subject.

## 4. Fee Portal
- Detailed breakdown of fee structures and historical receipts for parents.

## 5. Timetable Grid
- Visual calendar showing weekly class distributions.

## 6. Messaging Center
- Inbox-style interface for institutional communication.
