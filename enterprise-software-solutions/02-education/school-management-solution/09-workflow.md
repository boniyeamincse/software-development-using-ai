# 09 - System Workflows

## Student Admission Process
1. Parent fills out online application form.
2. System validates data and sends confirmation.
3. Admin reviews application and schedules interview/test.
4. Admission status updated to "Accepted".
5. Student details moved to SIS; Fee invoice generated.

## Examination Workflow
1. Teacher creates exam schedule and syllabus in the system.
2. Notification sent to students/parents.
3. Students appear for exam (online or physical).
4. Teacher uploads/marks grades.
5. System computes final results and ranks.
6. Principal approves; Report cards released digitally.

## Fee Payment Workflow
1. System generates monthly/termly invoice.
2. WhatsApp/SMS reminder sent to parents.
3. Parent pays via integrated gateway.
4. Receipt automatically generated and mailed.
5. Student's financial status updated in SIS.
