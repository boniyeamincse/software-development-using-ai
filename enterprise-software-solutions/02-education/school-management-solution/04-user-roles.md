# 04 - User Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Super Admin** | Full Access | System configuration, backup, user management. |
| **School Admin** | Operational Access | Managing admissions, fees, and staff records. |
| **Teacher** | Classroom Access | Attendance, grading, communication with students. |
| **Accountant** | Financial Access | Fee collection, payroll management, auditing. |
| **Librarian** | Asset Access | Book cataloging, issuing, and fine collection. |
| **Student** | Personal Access | Viewing grades, attendance, and assignments. |
| **Parent** | Guardian Access | Viewing ward's progress and fee payments. |
