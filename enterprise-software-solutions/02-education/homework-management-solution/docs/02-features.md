# 02 - Homework Core Features

## 1. Multi-Format Assignments
- Support for text, image, PDF, and video-based assignments.
- Integration with Google Drive and OneDrive for cloud submissions.

## 2. Automated Grading (Quizzes)
- Built-in MCQ and short-answer quiz engine for instant grading.
- Logic for "Partial Credit" and "Auto-Retake" on failed attempts.

## 3. Digital Classroom Whiteboard
- Teachers can record "How-to" clips or voice notes alongside the homework.
- Shared workspace for collaborative group assignments.

## 4. Submission Tracking & Deadlines
- Countdown timers for upcoming deadlines.
- "Late Submission" flagging with automated penalty calculations.

## 5. Rich Annotation Tools
- Teachers can draw, underline, and highlight directly on student-submitted PDFs or images.
- Voice-memo feedback for more personal teacher-student communication.

## 6. Smart Notifications
- Push notifications to students 24 hours before a deadline.
- Weekly summary emails to parents regarding completion rates.
