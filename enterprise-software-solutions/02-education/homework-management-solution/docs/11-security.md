# 11 - Homework Security & Privacy

## Student Safety
- **COPPA Compliance**: Strict adherence to children's online privacy protection.
- **Content Filtering**: Automatic scanning of uploaded images for inappropriate content using AWS Rekognition.

## Data Integrity
- **Virus Scanning**: ClamAV integration for all incoming student file uploads.
- **Immutable Audit Logs**: Tracking who accessed or edited a grade and when.

## Authentication
- **OAuth 2.0 Integration**: Support for "Login with Google/Microsoft for Education."
