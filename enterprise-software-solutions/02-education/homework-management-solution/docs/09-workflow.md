# 09 - Homework System Workflows

## Standard Assignment Lifecycle
1. **Creation**: Teacher creates homework, attaches resources, and sets the deadline.
2. **Distribution**: Notification sent to students and parents; Assignment appears in student dashboard.
3. **Execution**: Student views resources and works on the assignment.
4. **Submission**: Student uploads file; System records timestamp and validates file type.
5. **Grading**: Teacher reviews, annotates, and assigns a score.
6. **Release**: Grades made visible to students/parents on the "Release Date."

## Late Submission Handling
1. Deadline passes; Submission status changes to "Late."
2. System automatically applies a configurable penalty (e.g., -5% per day).
3. Notification sent to teacher flagging the late entry.
