# 10 - Homework Technology Stack

## Development Tools
- **Frontend**: React (Web), Flutter (Mobile).
- **Backend API**: Node.js/Express (Running on AWS Lambda).
- **Real-time**: Pusher or Socket.io.

## Data & Storage
- **Database**: PostgreSQL (Core Data) and DynamoDB (Status tracking).
- **Search**: Meilisearch for the Assignment Library.
- **Storage**: AWS S3 (Encrypted).

## Infrastructure
- **Hosting**: AWS.
- **CI/CD**: CircleCI.
- **Monitoring**: Datadog for tracing large file upload performance.
