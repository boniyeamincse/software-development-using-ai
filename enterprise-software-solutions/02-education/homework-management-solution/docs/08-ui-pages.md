# 08 - Homework UI Pages

## 1. Student "To-Do" Dashboard
- Gamified view with progress bars.
- Color-coded priority tags (e.g., "Due Today" in Red).

## 2. Interactive Assignment Creator
- Drag-and-drop builder for teachers to attach files and set scoring rubrics.

## 3. The "Focus" Viewer
- A clean, distraction-free reader for PDFs and videos inside the app.

## 4. Submission Inbox
- A list view for teachers with filters for "Graded" vs. "Pending."

## 5. Parent "Snapshot" View
- Simple weekly calendar view of their child's workload.
