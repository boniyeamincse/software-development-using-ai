# 09 - Scholarship Workflows

## The Selection Lifecycle
1. **Creation**: Admin sets up a fund and defines "Hard Rules" (GPA > 3.5).
2. **Announcement**: Portal automatically promotes the scholarship to eligible students.
3. **Drafting**: Student starts application; System auto-saves progress.
4. **Validation**: On submission, system queries SIS to verify GPA and identity.
5. **Selection**: Committee performs blind reviews and ranks final candidates.
6. **Awarding**: Financial Office confirms funds; Official award letter generated.
7. **Execution**: System triggers disbursement to the student’s tuition account.

## Renewal Workflow
1. End of Semester; System checks if recipient maintained GPA requirements.
2. If GPA is met, next disbursement scheduled.
3. If GPA is low, "Warning" sent to student and Counselor.
