# 10 - Scholarship Technology Stack

## Core Development
- **Frontend**: Next.js (Admin/Student) & Mobile Web (Responsive).
- **Backend API**: Python (FastAPI) or Node.js.
- **Workflow Engine**: Temporal.io or AWS Step Functions.

## Specialized Data
- **Primary DB**: PostgreSQL with Citus (for multi-tenant scalability).
- **Reporting**: Metabase or PowerBI integration.
- **OCR/Document Scanning**: Amazon Textract (to verify printed documents).

## Infrastructure
- **Cloud**: AWS or Azure.
- **Vault**: HashiCorp Vault for sensitive student records.
- **Compliance**: Vanta for continuous security monitoring.
