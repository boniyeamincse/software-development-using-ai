# Tutoring Marketplace Solution - Development Prompts

## 1. Example AI Delivery Prompts

1. Design the tutor onboarding domain model with verification states and audit fields.
2. Create booking APIs with idempotency and double-booking prevention.
3. Model tutor availability rules with recurring slots and blackout periods.
4. Design payout settlement logic with commission and refund adjustments.
5. Build a search ranking strategy for tutors using rating, response time, and subject fit.
6. Create guardian approval flows for minor learners.
7. Define the dispute lifecycle for no-show and refund cases.
8. Design analytics events for conversion funnel tracking.
9. Implement secure review submission rules tied to completed sessions.
10. Create an admin moderation queue for tutor verification and abuse reports.
