# Tutoring Marketplace Solution - Tech Stack

## 1. Recommended Stack

| Layer | Recommendation |
|---|---|
| Frontend Web | React with TypeScript |
| Mobile | React Native or Flutter |
| Backend | Node.js with NestJS or Java Spring Boot |
| Database | PostgreSQL |
| Cache | Redis |
| Search | OpenSearch or Elasticsearch |
| Queue | RabbitMQ or cloud queue service |
| Storage | S3-compatible object storage |
| Video | Zoom SDK, Twilio, or Daily |
| Payments | Stripe, Razorpay, or regional gateway |
| Monitoring | OpenTelemetry, Prometheus, Grafana |

## 2. Selection Priorities

- Fast delivery for an MVP
- Strong support for payments and integrations
- Good event-driven extension path
- Reliable search and analytics support
