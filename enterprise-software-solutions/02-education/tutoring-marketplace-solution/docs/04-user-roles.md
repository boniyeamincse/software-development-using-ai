# Tutoring Marketplace Solution - User Roles

## 1. Primary Roles

| Role | Description |
|---|---|
| Student | Books and attends tutoring sessions |
| Parent/Guardian | Manages dependent learners and payments |
| Tutor | Delivers tutoring services |
| Marketplace Admin | Operates the marketplace and configures policies |
| Support Agent | Resolves operational cases and disputes |
| Compliance Reviewer | Verifies documents and risk flags |
| Finance Officer | Oversees revenue, refunds, and payouts |
| Institution Coordinator | Manages organization-based tutoring programs |

## 2. Permission Highlights

- **Students**: Search tutors, book sessions, attend classes, rate tutors.
- **Parents/Guardians**: Approve bookings for minors, pay, review progress, manage learners.
- **Tutors**: Publish profiles, manage availability, conduct sessions, access earnings.
- **Admins**: Moderate users, manage categories, configure fees, view platform analytics.
- **Support Agents**: Handle tickets, booking issues, refund requests, and escalations.
- **Compliance Reviewers**: Approve or reject verification evidence and risk flags.
- **Finance Officers**: Reconcile settlements, monitor payout batches, export reports.
- **Institution Coordinators**: Assign learners, approve tutors, monitor service delivery.
