# 08 - Parent Communication UI Pages

## 1. Unified Inbox
- WhatsApp-style chat list with unread badges and snippet previews.
- Quick filters for "Admin Notices" vs. "Teacher Chats."

## 2. Parent-Teacher Scheduler
- Visual calendar grid where parents can tap an empty slot to book an interview.

## 3. The Digital Board (Feed)
- Infinite scroll of school updates, newsletters, and sports results.
- Embedded media support (Auto-playing silent videos).

## 4. Settings & Language
- Simple dropdown to change the primary language of the entire app experience.

## 5. Daily Log (Early Years)
- Visual icons showing a child's day (Happy face for good mood, bottle icon for meals).
