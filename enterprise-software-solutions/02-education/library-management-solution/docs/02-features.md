# 02 - Library Core Features

## 1. Advanced Cataloging & Search (OPAC)
- Support for MARC21 and Dublin Core metadata standards.
- Integrated search with "Faceted Filters" (Author, Format, Language, Publication Date).
- Full-text search for digital assets (PDFs, E-books).

## 2. RFID & Barcode Integration
- Rapid checkout/return using RFID gate and scanner integrations.
- Automated shelf-inventory using handheld RFID readers.

## 3. Circulation & Loan Management
- Flexible loan rules (Standard, Reference, High-Demand).
- Automated fine calculation based on user roles (Student vs. Faculty).
- "Hold" and "Reservation" queues with automated notification triggers.

## 4. Digital Repository Management
- Integrated viewer for E-books and DRM-protected journals.
- "Concurrent License" management for digital media.

## 5. Procurement & Acquisition
- Tools for librarians to track book orders, budgets, and vendor performance.
- Wishlist feature allowing students to suggest new titles.

## 6. Smart Notifications
- Multi-channel alerts (SMS, Email, Mobile Push) for overdue items and hold availability.
