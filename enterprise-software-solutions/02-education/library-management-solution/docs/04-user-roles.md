# 04 - Library Roles & Permissions

| Role | Access Level | Primary Responsibilities |
| :--- | :--- | :--- |
| **Head Librarian** | Full Access | Budgeting, procurement approval, staff management. |
| **Cataloger** | Metadata Access | Creating and editing book records (MARC/Dublin). |
| **Circulation Staff** | Operational | Checkout, returns, fine collection, hold management. |
| **Student / Member** | Standard Access | Searching, borrowing, paying fines, suggesting books. |
| **Guest** | Search-Only | Browsing the public catalog without borrowing. |
