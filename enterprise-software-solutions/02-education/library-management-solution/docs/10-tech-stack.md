# 10 - Library Technology Stack

## Development Tools
- **Backend**: Python (Django REST Framework).
- **Frontend**: React + TypeScript (Ant Design for Librarian UI).
- **Search Engine**: Elasticsearch / Kibana.

## Data & Storage
- **Database**: PostgreSQL (Relational) + Redis (Session Cache).
- **File Store**: MinIO or AWS S3 (for E-books).

## Hardware & IoT
- **Protocols**: LLRP (Low-Level Reader Protocol) for RFID integration.
- **Scanning**: Honeywell/Zebra Barcode API support.

## Infrastructure
- **Hosting**: DigitalOcean or AWS.
- **CI/CD**: GitHub Actions.
